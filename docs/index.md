# Cravia Healthcare


## **Problem Statement**
This project aims to develop an online healthcare management platform that allows patients to book appointments easily, choose their preferred doctor, time, and location, and receive real-time updates on their booking status 

## **Solution Overview**
The Healthcare Management System solves these problems by providing a centralized digital platform where:   

- Find verified doctors across specializations with details like experince.    
- Book appointment with doctors according to health condition.   
- Clinic and slot-based doctor appointment booking.     


This system will manage the bridge gap between paitents, doctors and healthcare education by making quality healthcare accessible, informed, and personalized for everyone.  

 
---

## **Key Features**

-  Doctor listing with details (specialization, experience)
-  Clinic and slot-based doctor appointment booking 
-  Online appointment booking 
-  Secure user authentication (Sign-up / Login)

---

# **Technology Stack**  

## **Back-End**
- Framework:Node.js with Express(REST API). 
- Authentication:JWT (JSON Web Tokens). 
- Database:POSTGRESQL. 
- Encryption:bcrypt for password hashing.  

## **Front-End**  
- Framework:React.
