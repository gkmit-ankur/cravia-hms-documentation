# Tech Stack

## Frontend

### React
- A JavaScript library for building user interfaces.
- Enables the creation of dynamic and responsive web applications.

## Backend

### Node.js
- A JavaScript runtime built on Chrome's V8 engine.
- Supports high-performance server-side development.

### Express.js
- A minimal and flexible Node.js web application framework.
- Provides robust features for building APIs.

## Database

### RDS
- Amazon Relational Database Service.
- Simplifies setup, operation, and scaling of relational databases in the cloud.

### PostgreSQL
- A powerful, open-source relational database system.
- Known for reliability and a rich feature set.

### Sequelize
- An ORM (Object-Relational Mapping) tool.
- Simplifies database access, modeling, and management.

## DevOps

### AWS
- Cloud platform offering scalable databases and deployment services.
- Ensures reliable application hosting and operations.

## Testing

### Jest
- A simple, modern JavaScript testing framework.
- Designed for large-scale web applications with a focus on ease of use.
